    <article class="markdown-body entry-content" itemprop="text"><h1><a id="user-content-homework-3---mnist-handwritten-digit-recognition" class="anchor" aria-hidden="true" href="#homework-3---mnist-handwritten-digit-recognition"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Homework 3 - MNIST handwritten digit recognition</h1>

<li>The goal of this homework is to make you familiar with applying some machine learning algorithms using Python.</li>

<ul>
<li><a href="https://www.python.org/download/releases/3.0/" rel="nofollow">Python3</a></li>
<li><a href="http://scikit-learn.org/stable/" rel="nofollow">scikit-learn</a> (You can use other machine learning libraries)</li>
</ul>
<h2><a id="user-content-dataset" class="anchor" aria-hidden="true" href="#dataset"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Dataset</h2>
<ul>
<li>Download MNIST dataset from <a href="http://yann.lecun.com/exdb/mnist/" rel="nofollow">here</a>. Do not use sklearn built-in functions to load the dataset, as it uses different training and testing set as the website. During evaluation, we will assume the results corresponds to the testing set from the website.</li>
<li>Use only the training images for training (You can split it into training set and validation set).</li>
<li>The test set will be used for evaluation.</li>
</ul>
<h2><a id="user-content-assignment-description" class="anchor" aria-hidden="true" href="#assignment-description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Assignment Description</h2>
<ul>
<li>Apply logistic regression and random forest on MNIST.</li>
<li>For both approaches, use the raw pixel values as features.</li>
</ul>

A sample submission result can be found <a href="/qge2/ece542-csc591-2019spring/blob/master/docs/samples/sample_submission">here</a>.</li>

</ul>
